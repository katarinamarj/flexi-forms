# flexi-forms
This is a web application developed using PHP Symfony for the backend and React for the frontend. The application enables creating, editing, and managing user-defined forms.

2. Login and registration
   
The login page allows users to access the application by entering their username and password.
The registration page enables new users to create an account by providing a username, password, full name, and email address.

3. My Templates page
   
The My Templates page allows users to view all the forms they have created. From this page, users can delete existing forms, copy a shareable link, or view the submitted responses for each form.
<img width="1920" height="1353" alt="Image" src="https://github.com/user-attachments/assets/cfe7d547-8995-488b-810e-9f5289988c58" />

4. Create New Form page
   
The Create New Form page consists of three steps. In the first step, the user enters the form name and description. In the second step, the user adds the form fields. In the final step, the user reviews the details and confirms the creation of the form.

5. Responses page
   
The Responses page provides two ways to view submitted answers. The summary view displays aggregated responses for each question, while the individual view shows the complete set of answers submitted by each respondent separately.
